Quero que você atue como um consultor de carreira e crie um inventário completo de habilidades. 
O inventário deve estar organizado em categorias: 
- Habilidades técnicas 
- Habilidades comportamentais 
- Habilidades criativas 
- Habilidades acadêmicas 
- Habilidades profissionais

Para cada categoria:
1. Liste de forma estruturada as principais habilidades relevantes para o desenvolvimento pessoal e profissional.  
2. Inclua exemplos práticos de como cada habilidade pode ser aplicada em um contexto de carreira.  
3. Sugira níveis de proficiência (iniciante, intermediário, avançado) e possíveis caminhos de evolução.  
4. Mostre como essas habilidades podem se conectar entre si para formar um perfil completo.  

O inventário deve ser apresentado em formato claro, organizado e motivador, com linguagem acessível e inspiradora, para que eu possa usar como exercício pessoal de autoconhecimento e planejamento de carreira.
